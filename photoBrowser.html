Instructions: This application loads photos from a specified folder using the base file name and a simple numbering system that starts at zero. It only displays photo files that end in *.jpg. For example, if you specify a folder name of "images", a base file name of "leaf", and the number of photos to browse as "3", the application will load three files to browse: "images/leaf0.jpg", "images/leaf1.jpg", and "images/leaf2.jpg".
